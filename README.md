# ku-cli-python
Very simple KuCoin cli. 

Create these in the KuCoin web portal and set them in your shell:

```
export API_KEY=
export API_SECRET=
export API_PASSPHRASE=
```

## Notes
Remember to whitelist withdrawals and IP access.
